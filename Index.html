Cyber Safety Tips

🎭
Spot Deepfakes
Look for unnatural eye blinking, lighting inconsistencies, blurry edges near face, and audio-video sync issues. Always verify via official channels.
📧
Avoid Phishing
Never click links in unexpected emails. Check sender domain carefully. Legitimate banks never ask OTP via email. Use official apps only.
📶
WiFi Safety
Avoid public WiFi for banking. Use VPN. Check if network uses WPA3. Disable auto-connect. Watch for duplicate network names (evil twin attacks).
🏦
UPI Fraud Prevention
Never share UPI PIN. Receiving money doesn't require PIN entry. Verify caller identity. Report fraud immediately via 1930 helpline.
📱
Safe APK Practices
Only install from Play Store / App Store. Avoid APKs from WhatsApp/Telegram. Check permissions before installing. Revoke unused permissions.
🔑
Strong Passwords
Use 12+ characters with symbols. Never reuse passwords. Enable 2FA everywhere. Use a password manager. Change passwords every 90 days.
